Learning Recommendation System

AI-powered system that recommends personalized courses to students based on their learning behavior.

Features:
- Personalized course recommendations using Machine Learning
- Student profile analysis with statistics
- Interactive data visualizations
- Professional GUI interface
- Evaluation metrics (Precision@K, Recall@K)

How to Run

1. Install dependencies: pip install -r requirements.txt
2. Run the application: python rec.py


Tech Stack
- Python 3.11
- Machine Learning: scikit-learn (TF-IDF, Cosine Similarity)
- Data Processing: Pandas, NumPy
- GUI: Tkinter
- Visualization: Matplotlib

Project Structure
- rec.py - Main application with ML model and GUI
- requirements.txt - Python dependencies
- README.MD - About the project 


Dataset
- 30 diverse courses across 8 categories
- 20 students with realistic interaction patterns
- Synthetic data generated for demonstration
